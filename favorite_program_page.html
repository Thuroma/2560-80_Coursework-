My favorite Program!

My favorite program I have written so far was my forst program controlling my Tello drone. This was the first time I wrote code that was used to power another device.

This code snippet:

  • 1. Connects to the drone
  • 2. Displays the battery on the video output
  • 3. Lifts the drone
  • 4. Turns it around
  • 5. Flys it forward
  • 6. Lands the drone
                    
from djitellopy import tello
import cv2
from time import sleep

PORT = 152
drone = tello.Tello()
drone.connect()
print(drone.get_battery())

drone.takeoff()
drone.rotate_clockwise(180)
drone.move_forward(25)
sleep(2)

drone.land()
